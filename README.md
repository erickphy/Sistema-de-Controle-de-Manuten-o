Caso: A "InfoTech IFPB" precisa de um site simples para que clientes possam ver os serviços oferecidos e abrir um chamado de manutenção.
Usuários-alvo:
Clientes: Querem saber o que a empresa faz e pedir um serviço.
Técnicos: (Futuramente) Usariam uma área interna para ver os chamados (Não faremos o login, apenas a página).
Funcionalidades (Estáticas):
Home: Apresentação da empresa e destaque dos serviços.
Serviços: Página com "cards" detalhando os serviços (Hardware, Software, Redes).
Contato/Chamado: Formulário para abrir um novo chamado de manutenção.
